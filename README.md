 List of features :

- **Paginated Coin List/Grid** 📋: Display a paginated list or grid of cryptocurrencies. Each page contains a specified number (20 items) of items with navigation to load more.
<img width="1427" alt="Screenshot 2024-07-20 at 6 57 45 PM" src="https://github.com/user-attachments/assets/eee6e311-1070-4e01-a6d2-24e9be5b8764">

- **Navigation** 🔗: Clicking on a card routes the user to the product page of the selected cryptocurrency.
  <img width="1433" alt="Screenshot 2024-07-20 at 6 58 11 PM" src="https://github.com/user-attachments/assets/54e6a608-394a-41bd-b8fc-de70c6f6793a">

- **Product Page** 📈:
  - Display basic information about the selected cryptocurrency.
  - Show a candle/line graph of the cryptocurrency’s price over time.

- **Draggable Watchlist** 🧩: A drag-and-drop functionality that allows users to easily add coins to their watchlist by dragging and dropping them.

- - **Search Bar with Suggestions** 🔍: Integrate a feature in the search bar that displays recently searched items as suggested items.
 <img width="1435" alt="Screenshot 2024-07-20 at 7 01 24 PM" src="https://github.com/user-attachments/assets/dc0c8eb8-e758-4a12-b2e2-74597ec1d2f1">

- **Reliable Cryptocurrency API** 🌐:  A reliable API ( CoinGecko API) to fetch real-time and historical data for cryptocurrencies.

- **Dynamic Theme Switching** 🌗:  A dynamic theme-switching feature that allows users to toggle between Light and Dark modes seamlessly.
- <img width="1440" alt="Screenshot 2024-07-20 at 7 02 00 PM" src="https://github.com/user-attachments/assets/8fbc6fb4-7285-4d37-a790-46c8e625fbe5">

- **Server-Side Rendering with Next.js** ⚙️: Utilize Next.js's server-side rendering capabilities for improved performance and SEO.
